
# EXERCICE DEFINITIONS

Objectif : Définir des termes basiques

## Définitions

1. **HTTP** : protocole
2. **GET**: appartient à HTTP, c'est des méthodes quand on veut récupérer quelque chose exemples : POST, UPDATE, DELETE, PUT, HEAD
3. **DNS** : service
4. **Https://** : s pour secure, protocole de début d'URL
5. **URL** : Unique Ressources Locator
6. **Dummyjson.com** : nom de doaine, top level domaine fourni par IAANA, il peut y avoir des sous domaine
7. **443** : port pour https, 80 c'est pour HTTP
8. **recipes** : nom de la ressources
9. **HTTP/2** : version du protocole
10. **? ; limit = 10** : le ? est un délimiteur, tout ce qui est après n'est pas une ressource mais des dictionnaires. Limite = paramètre de la ressources
